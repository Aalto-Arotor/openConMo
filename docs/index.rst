OpenConMo: Open-Source Software for Vibration Based Condition Monitoring
========================================================================

OpenConMo is a Python library for vibration signal based condition monitoring developed at Aalto University, Finland. The objectives of this library are as follows:

1. Provide easy access to reproducing signal based condition monitoring papers
2. Enable comparison of AI/ML based techniques with conventional signal processing tools

OpenConMo GitHub page: https://github.com/Aalto-Arotor/openConMo.git
